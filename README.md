# 🤖 News Summary Automation

> **RSS + Make + OpenAI + Notion을 활용한 뉴스 요약 자동화 시스템**

---

# 📌 프로젝트 한눈에 보기

> **AI/IT 뉴스를 자동으로 수집 → AI가 요약 → Notion에 저장하는 자동화
> 프로젝트**

## 🎯 왜 만들었나요?

매일 새로운 AI 뉴스를 찾아보고 정리하는 과정은 반복적이고 시간이 많이
소요됩니다.

이 프로젝트는 **RSS + Make + OpenAI + Notion**을 연결하여 사람이 직접
복사·붙여넣기 하지 않아도 뉴스를 자동으로 관리할 수 있도록 만들었습니다.


### 🎯 프로젝트 목표

- RSS를 이용하여 최신 AI 뉴스를 자동으로 수집한다.
- AI를 활용하여 뉴스를 핵심 내용만 요약한다.
- 사람이 복사·붙여넣기 하지 않아도 Notion에 자동 저장한다.
- 동일한 뉴스가 여러 번 저장되지 않도록 중복을 방지한다.

# ✨ 주요 기능 (Features)

## 📰 RSS 자동 수집

**무엇을 하나요?**

RSS Feed에서 최신 AI 뉴스를 자동으로 가져옵니다.

**왜 필요한가요?**

사람이 뉴스 사이트를 매번 방문하지 않아도 최신 뉴스를 자동으로 수집할 수 있습니다.

---

## 🏷️ 주제 필터링

**무엇을 하나요?**

AI 관련 키워드를 포함한 기사만 선택합니다.

**왜 필요한가요?**

원하지 않는 일반 뉴스는 제외하고 필요한 기사만 자동으로 가져오기 위해 사용합니다.

**사용 키워드** (예시)

- AI Agent
- LLM
- Generative AI
- OpenAI
- Claude
- Gemini
- Automation
- Enterprise AI

---

## 🤖 AI 자동 요약

**무엇을 하나요?**

OpenAI (또는 gemini, openrouter, graq)를 이용하여 기사 내용을 3줄 이내로 요약합니다.

**왜 필요한가요?**

긴 기사를 모두 읽지 않아도 핵심 내용을 빠르게 확인할 수 있습니다.

---

## 📦 JSON 변환

**무엇을 하나요?**

AI가 생성한 결과를 Notion에서 저장할 수 있는 형식(JSON)으로 변환합니다.

**왜 필요한가요?**

Notion은 정해진 데이터 형식으로 정보를 받아야 자동 저장이 가능합니다.

---

## 🔍 URL 중복 검사

**무엇을 하나요?**

기사의 URL을 기준으로 이미 저장된 기사인지 확인합니다.

**왜 필요한가요?**

같은 뉴스가 여러 번 저장되는 것을 방지하기 위해 사용합니다.

---

## 📝 Notion 자동 저장

**무엇을 하나요?**

최종 결과를 Notion 데이터베이스에 자동 저장합니다.

**저장 항목**

- 제목
- AI 요약
- 원문 링크
- 발행일

**왜 필요한가요?**

뉴스를 한곳에서 쉽게 검색하고 관리할 수 있습니다.

---

## 🔄 예외 처리 (Retry)

**무엇을 하나요?**

OpenAI 또는 Notion 호출이 실패하면 일정 시간 후 다시 시도합니다.

**현재 적용 정책**

| 상황 | 처리 방식 |
|------|-----------|
| OpenAI 호출 실패 | 10분 후 재시도 |
| Notion 저장 실패 | 10분 후 재시도 |
| 최대 재시도 횟수 | 2회 |

**왜 필요한가요?**

외부 서비스는 일시적으로 실패할 수 있기 때문에 자동으로 다시 시도하여 안정성을 높입니다.

# 💭 이번 프로젝트를 통해 배운 점

-   RSS는 뉴스를 자동으로 가져오는 역할을 한다. (API 호출 방법 외)
-   Make는 여러 서비스를 연결하여 자동화를 수행한다.
-   OpenAI는 뉴스 내용을 요약하는 역할을 한다.
-   Notion은 자동으로 정리된 데이터를 저장하는 공간이다.
-   자동화는 하나의 프로그램이 아니라 여러 서비스를 연결해 하나의
    흐름(Workflow)을 만드는 과정이라는 점을 이해했다.

# 📖 용어 한눈에 보기

| 용어 | 쉽게 설명 |
|------|-----------|
| RSS | 뉴스가 새로 올라오면 자동으로 알려주는 목록 |
| Workflow | 자동으로 실행되는 작업 순서 |
| Make | 여러 서비스를 연결해 자동화를 만드는 도구 |
| OpenAI | 기사 내용을 요약하는 AI |
| JSON | 프로그램끼리 데이터를 주고받는 형식 |
| Notion DB | 데이터를 표 형태로 저장하는 공간 |
| Retry | 실패했을 때 자동으로 다시 시도하는 기능 |
